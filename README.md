# DomAIyn Lab

## AI-Powered Prompt Security and Threat Detection System

PromptShield AI is a deployed AI based system designed to analyze user prompts and identify potentially unsafe, malicious or policy-violating inputs in real time.

The system acts as a lightweight defensive layer for AI applications by classifying prompts into safe and unsafe categories, enabling safer interaction with language models.

------------------------------

## Live Application

Access the deployed system here:
https://promptshieldai.streamlit.app/

------------------------------


## Overview

Modern AI systems are increasingly vulnerable to prompt-based attacks, including jailbreak attempts, harmful instructions and misuse scenarios.

DomAIyn Lab addresses this by introducing a real-time prompt analysis layer that evaluates user inputs before they are processed further. The goal is to provide an additional safety checkpoint for AI-driven applications.

------------------------------


## Core Functionality

* Real-time prompt analysis
* Classification of inputs as safe or unsafe
* Immediate visual feedback through an interactive interface
* Lightweight and deployable architecture
* Designed for integration as a pre-processing layer in AI systems

------------------------------


## System Workflow

1. User submits a prompt
2. The system analyzes the input using predefined logic and/or AI-based evaluation
3. A classification decision is generated
4. The result is displayed with corresponding feedback

------------------------------


## Use Cases

* AI application safety filtering
* Prompt validation before LLM processing
* Educational demonstrations of prompt security
* Hackathon prototypes for AI safety systems

------------------------------


## Tech Stack

* Python
* Streamlit
* Basic NLP / rule-based analysis

------------------------------


## Limitations

* The system uses simplified detection logic and is not production-grade
* Does not include large-scale threat intelligence or adaptive learning
* Limited handling of complex adversarial prompts

------------------------------


## Future Improvements

* Integration with advanced language models
* Context-aware threat detection
* Scoring-based risk analysis instead of binary classification
* API layer for external integration
* Expanded dataset for better detection accuracy
